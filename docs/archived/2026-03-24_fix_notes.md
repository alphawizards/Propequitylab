## Issue: Site Down — Blank White Screen in Production

### Root Cause
DNS for `propequitylab.com` points to a **Cloudflare Pages** project (`propequitylab.pages.dev`), but the environment variables (including `REACT_APP_CLERK_PUBLISHABLE_KEY`) are configured on a separate **Cloudflare Workers** project. The Pages project doesn't have the Clerk key, so the app crashes on boot.
